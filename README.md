# Infogenius

Generate clean, shareable infographics from your prompts using Gemini.

## Features

- Prompt-driven infographic generation
- Search results view with rich cards
- Clean, fast React + Vite UI

## Tech Stack

- React 19
- Vite 6
- TypeScript
- Gemini API

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Scripts

- `npm run dev` - start the dev server
- `npm run build` - create a production build
- `npm run preview` - preview the production build
